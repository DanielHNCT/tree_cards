La vie d'Ada

Lovelace, mathématicienne britannique du XIXe siècle, est considérée comme la première programmeuse de l'histoire pour avoir écrit le premier algorithme destiné à être exécuté par une machine, la machine analytique de charles babbage, ce qui en fait une pionnière visionnaire de l'informatique moderne. Le nom de lécole Ada Tech School s'inspire de cette mathématicienne.

En quelques points :

  • Pionnière de l'informatique : créa le prmier algorithme pour ordinateur
  • Mathématicienne visionnaire du 19e siècle
  • Collaboratrice de Babbage sur la machine analytique
  • Anticipait le potenciel des ordinateurs au-delà des simples calculs
